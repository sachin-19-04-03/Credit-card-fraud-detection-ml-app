# 💳 Credit Card Fraud Detection System

An end-to-end **Credit Card Fraud Detection ML application** built using **XGBoost, SMOTE, Flask API, and Streamlit** for real-time transaction risk scoring and fraud prediction.

---

## 🚀 Project Overview
This project detects fraudulent credit card transactions using advanced machine learning techniques.

It includes:
- 📊 Exploratory Data Analysis (EDA)
- 🛠 Feature Engineering
- 🤖 XGBoost + SMOTE Retraining
- 🌐 Flask REST API
- ⚡ Thunder Client API Testing
- 🎨 Streamlit Interactive Dashboard
- 🚨 Real-time Fraud Prediction

---
